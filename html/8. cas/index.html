Inline elementi

Inline elementi zauzimaju samo potreban prostor.

Ovi elementi ne mogu imati width i height.

NAPOMENA. Unutar Inline elemenata child moze biti samo Inline element, dok unutar Block level elemenata child moze biti i Block level element i Inline element.
